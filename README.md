# ZvládnuVýšku

Neoficiální studentský web pro Ostravskou univerzitu.

ZvládnuVýšku vzniklo z praktického problému: před zapsáním předmětu většinou chybí jedno místo, kde by šlo rychle zjistit reálnou obtížnost, časovou náročnost, zkušenosti s vyučujícími, dostupné materiály a možnost si látku rovnou procvičit. Oficiální sylabus tohle nepokrývá. Cílem projektu je dát studentům použitelné informace od studentů, ne jen formální popis předmětu.

Projekt nemá žádnou oficiální spojitost s Ostravskou univerzitou ani s jednotlivými fakultami.

## Co Je ZvládnuVýšku

Webový studentský hub, který spojuje:

- vyhledávání předmětů podle názvu a zkratky
- hodnocení předmětů od studentů
- přehled vyučujících a jejich hodnocení
- veřejné studijní materiály
- kartičky a procvičování se spaced repetition
- přehlednější rozhodování před zápisem předmětu

## Hlavní funkce

### Předměty
- fulltextové vyhledávání předmětů
- přehled kreditu, semestru, docházky, obtížnosti a časové náročnosti
- detail předmětu s praktickými informacemi a studentským hodnocením

### Vyučující
- katalog vyučujících podle fakult
- detail vyučujícího
- studentské hodnocení a komentáře
- návrhy nových vyučujících ke schválení

### Materiály
- veřejný katalog schválených studijních materiálů
- vyhledávání napříč materiály
- přiřazení materiálů k předmětům

### Kartičky
- veřejné i soukromé balíčky kartiček
- přiřazení balíčku ke konkrétnímu předmětu
- create/edit flow pro vlastní balíčky
- více typů otázek:
  - klasická kartička
  - multiple choice
  - ano/ne
  - otevřená odpověď
- obrázek u otázky
- procvičování se spaced repetition

### Chytré vyhledávání
- běžné hledání předmětů
- `.f` režim pro kartičky
- `.m` režim pro materiály

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Supabase

## Poznámka k repozitáři

Repozitář je veřejný kvůli transparentnosti a dokumentaci vývoje. Není vedený jako obecný template ani jako hotový white-label produkt.
