# Frontend

Frontend aplikace ZvládnuVýšku postavený na Next.js.

Obsahuje:

- landing page a jednotné vyhledávání
- katalog předmětů
- katalog vyučujících
- katalog materiálů
- sekci kartiček a procvičování
- administrační rozhraní pro schvalování a správu obsahu

Hlavní app entrypoint je v `app/`.
